# Lombok 이란?
Lombok 은 자바에서 보일러플레이트 코드를 줄여주는 라이브러리입니다. 예를 들어 getter/setter, equals, hashCode, toString 메서드와 같은 반복적인 코드를 자동으로 생성할 수 있어 코드가 더 깔끔하고 간결해집니다. 이로 인해 유지보수가 쉬워지고 코드의 가독성이 높아집니다.
